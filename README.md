# Node.js Backend Learning Journey

This repository contains a structured series of tutorials and projects covering backend development with Node.js, Express, MongoDB, and more.

## Course Structure

1. **[Basics](./1.Basics)**
   - Introduction to core Node.js concepts and fundamental modules.
2. **[Express Tutorial](./2.Express-Tutorial)**
   - Learning how to build web servers and handle routing using the Express.js framework.
3. **[EJS Tutorial](./3.Ejs-Tutorial)**
   - Working with Embedded JavaScript (EJS) templating engine to render dynamic views.
4. **[REST API Tutorial](./4.Rest-api-tutorial)**
   - Fundamentals of building RESTful APIs, handling HTTP methods, and structuring API endpoints.
5. **[MongoDB Basics](./5.Mangodb-basics)**
   - Introduction to NoSQL databases with MongoDB and Mongoose.
6. **[Book Store API Development](./6.Book-store-api-development)**
   - A practical project building a Book Store API to apply REST and database concepts.
7. **[Node.js Authentication](./7.Nodejs-auth)**
   - Implementing user authentication and authorization (e.g., using JWT or sessions) in Node.js applications.
